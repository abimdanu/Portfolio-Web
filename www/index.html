Network Simulation with GNS3

GNS3 is a network software emulator which allows he combination of virtual and real devices. GNS3 can be used to simulate complex networks with Dynamips emulation to simulate Cisco IOS.

In this project, I was able to configure various routing protocols into simulation networks (Static Routing, RIP, EIGRP, OSPF, IS-IS, Multi-protocol BGP), subnetting, and switching in GNS3. The configuration steps was documented neatly into a 'manual' dcoument.
